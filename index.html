Security & Access Control

Five-tier access.
Zero compromise.

Every user has exactly the permissions their role requires. Access is invite-only, auditable, and enforced at the API level — not just the UI.

Super AdminFull platform access — manages organisations, system-wide configuration, and all users across the entire instance.
Org AdminManages all projects and members within an organisation — creates projects, invites team members, and configures org-level settings.
Project ManagerFull control within assigned projects — starts sprints, manages workflows, assigns issues, and configures project settings.
MemberCreates and updates issues, logs comments, and moves work through boards and backlogs day-to-day.
ViewerRead-only access for stakeholders and clients — complete visibility, zero risk of accidental changes.
Architecture
Multi-tenant data isolation
Every organisation's data is completely isolated at the database level. No cross-tenant leakage is architecturally possible — even for platform administrators.
Onboarding
Invite-only member access
New members join only via explicit email invitation. No open registration — every user is intentional, revocable, and fully auditable at any time.
Compliance
Complete audit trail
Every status change, comment, assignment, and permission update is logged with a timestamp and user ID — a full, queryable activity trail for compliance needs.
