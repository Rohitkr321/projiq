New Improved Security
July 9, 2026

Google Sign-In, Custom Fields Security & Performance

This release adds Google OAuth sign-in to the registration and login flows, hardens custom field permissions with project-scoped authorization, and fixes several performance and data correctness issues in the reporting layer.

  • Google Sign-In added to the registration and login screens — one tap to join or return
  • Custom fields now enforce project-scoped ownership checks — a project manager in Project A can no longer mutate fields belonging to Project B
  • Custom field routes now require role-based authorization at the route level (requireProjectRole), preventing unauthorized reads and writes
  • Project overview report now correctly counts open issues (unresolved) vs all issues — the counts were previously identical due to a duplicated query
  • Issue dependency link endpoint corrected — previously resolving to a non-existent route
  • Subscription management UI in Profile screen now shows plan limits, current usage, and upgrade prompts
  • Plan tier enforcement tightened — Starter org project and member caps now applied consistently at creation time

New Improved Fix
June 18, 2026

Kanban Swimlanes, Cumulative Flow Diagram & Roadmap View

The board now supports swimlane grouping by assignee or epic. The Reports tab gains a cumulative flow diagram for tracking WIP trends. The Timeline view has been promoted to a first-class navigation item.

  • Kanban swimlanes — group cards by assignee, epic, or priority with a single click
  • Cumulative Flow Diagram in the Reports tab — 30, 60, and 90-day windows
  • Timeline (Gantt) view promoted from beta — accessible from the main project nav
  • Sprint Planning modal now shows team capacity alongside story point totals
  • Backlog drag-and-drop re-ordering is now persisted immediately across sessions
  • Fixed an issue where completed sprints showed incorrect velocity when issues were moved post-completion

New Improved Fix
May 5, 2026

Time Tracking, Burndown Charts & Multi-Tenant Billing

Time logging arrives on issues with a dedicated Time Tracking tab. The Reports module launches with burndown and velocity charts. Billing moves from a shared model to per-organization subscriptions with three plan tiers.

  • Time Tracking — log time directly on any issue, view a per-user breakdown in Reports
  • Burndown chart in the Sprint Reports tab — ideal vs actual remaining story points
  • Three-tier subscription plans: Starter (free), Pro ($29/mo flat), Enterprise (custom)
  • Organization admin billing dashboard — view current plan, usage, and upgrade
  • Issue detail drawer redesigned — tabs for Details, Activity, Time, Dependencies
  • Real-time notifications now appear in-app via Socket.io push — no refresh needed
  • Fixed race condition where two users editing the same issue simultaneously could lose one set of changes
  • Sprint end date validation now correctly rejects end dates earlier than start dates

New Improved
March 20, 2026

Custom Fields, Issue Dependencies & Role-Based Access

Project managers can now define custom fields on any issue type. Issue dependencies (blocks / blocked by) let teams model sequencing. A full RBAC system with five roles replaces the previous two-level admin model.

  • Custom Fields — text, number, dropdown, date, and checkbox types; configurable per project
  • Issue dependencies — link issues as blocks / blocked by / relates to / duplicates
  • Five-role RBAC: super_admin, org_admin, project_manager, developer, reporter
  • Project-level member management — invite by email, assign roles per project
  • Issue search now supports full-text across title, description, and custom field values
  • Activity log on each issue records all field changes with before/after values

Projiq Launches — Sprint Boards, Kanban & Real-Time Collaboration

Projiq is live. Engineering teams can now manage sprints, run Kanban workflows, collaborate in real time, and invite their whole organization under one flat subscription — no per-seat fees.

  • Sprint board with backlog, active sprint, and completion workflow
  • Kanban board with customizable workflow columns and WIP limits
  • Real-time collaboration via Socket.io — see teammates' changes instantly
  • Multi-project workspace with organization-level user management
  • Issue types: Story, Bug, Task, Epic — with priority, assignee, labels, and due dates
  • Mobile app for iOS and Android — React Native, works on all screen sizes
  • Flat-price billing — unlimited members, one organization, one price
